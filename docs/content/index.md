<div class="feature">

### Fast Builds

Go-powered parallel pipeline. Thousands of pages in seconds, not minutes.

</div>

<div class="feature">

### Liquid Templates

Familiar template syntax with zero learning curve. No Go templates required.

</div>

<div class="feature">

### Tiered Plugins

QuickJS for lightweight transforms, WASM for performance-critical work, Node for the full npm ecosystem. Pick the right tool for the job.

</div>

<div class="feature">

### Data Cascade

Directory-level `_data.yaml` files set defaults that cascade to every page beneath them. Deep merge, no boilerplate.

</div>

<div class="feature">

### Pagination

Generate virtual pages from any data source — collection items, data files, API responses. One template, unlimited output.

</div>

<div class="feature">

### SSR

Server-side render web components at build time. Ship static HTML with full component fidelity.

</div>
